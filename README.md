# L'AURA – Modern Fine Dining

Live Demo :- https://srilehkakada-dot.github.io/FUTURE_FS_03/

**Culinary Artistry Redefined**

A responsive website for L'AURA, a modern fine dining restaurant focused on gastronomy & elegance. This project showcases the restaurant's ambiance, menu, and allows customers to book tables online.

### ✨ Features
- **Responsive Design**: Looks great on desktop, tablet, and mobile
- **Interactive Sections**: Home, About, Menu, Gallery, Contact
- **Online Reservations**: Form handler for booking tables
- **Contact Form**: Placeholder for customer inquiries  
- **Modern UI**: Elegant typography and smooth scroll navigation
